# guide-indexs-lejing

## 项目简介

本项目是一个用于归档和发布多个独立 HTML 页面的仓库。这些页面各自独立，不依赖特定域名或外部网站，主要用于存放静态内容或前端演示。

仓库本身不提供任何具体服务或功能，仅作为 HTML 文件的集合与版本管理载体。

## 目录说明

```
guide-indexs-lejing/
├── pages/           # 存放主要 HTML 页面文件
├── assets/          # 页面所需的资源文件（图片、样式、脚本等）
├── docs/            # 相关文档或说明
└── README.md        # 本文件
```

- `pages/`：各独立 HTML 页面的存放目录，每个文件可单独打开使用。
- `assets/`：页面引用的公共资源，可按需组织子目录。
- `docs/`：存放项目相关的附加文档或使用说明。

## 页面归档说明

- 所有 HTML 页面均为独立文件，可直接在浏览器中打开运行。
- 页面之间无依赖关系，可单独维护或更新。
- 归档内容可能不定期更新，具体变更请参考 Git 提交记录。

## 维护说明

- 欢迎提交 Issue 或 Pull Request 对页面内容进行修正或补充。
- 提交新页面时，请将 HTML 文件放入 `pages/` 目录，并确保资源引用路径正确。
- 建议在提交时附带简要说明，方便其他贡献者了解页面用途。
